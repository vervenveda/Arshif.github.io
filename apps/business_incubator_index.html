How the Incubator Works

The engine simulates many possible founder priorities, then looks for resources that repeatedly rise to the top.

1. Intake

Enter your idea, stage, industry, budget, model, and areas where you need help.

2. Monte Carlo-style scoring

The app runs repeated randomized scoring passes. Each pass gives slightly different weight to your stage, budget, industry, identity flags, needs, and customer type. Resources that keep ranking highly receive stronger confidence scores.

3. Pathway report

You receive a recommended action plan, top resource cards, and a founder checklist. You can save your profile locally, export a JSON backup, or print the plan.

4. Sovereign sandbox rules

This file contains all app logic inside one HTML file. It uses no CDN, no external libraries, no analytics, and no server calls. Links open only when clicked.
